About Yahoo Financeâ„¢ for Google Sheetsâ„¢

Yahoo Financeâ„¢ for Google Sheetsâ„¢ is a Google Workspace add-on that allows users to fetch public financial market data from Yahoo Financeâ„¢ directly inside Google Sheetsâ„¢.

The add-on provides simple custom functions such as YF_PRICE, YF_CURRENCY, YF_EXCHANGE, and YF_NAME to retrieve market prices and instrument metadata for stocks, ETFs, indices, and forex pairs.

This add-on is designed to be lightweight, read-only, and focused on providing up-to-date financial data without collecting or storing personal user information.

How It Works

After installing the add-on, you can use custom spreadsheet functions to retrieve public market data directly inside Google Sheetsâ„¢.

=YF_PRICE("AAPL")

Returns the current market price of the selected instrument.

=YF_CURRENCY("AAPL")

Returns the trading currency (for example USD or EUR).

=YF_EXCHANGE("AAPL")

Returns the full name of the exchange where the instrument is traded.

=YF_NAME("AAPL")

Returns the full or short name of the instrument.

Technical & Privacy Information

The add-on automatically manages caching to keep prices reasonably up to date while reducing external requests. Price data is cached for a short time, while instrument metadata such as name, exchange, and currency is cached for longer periods.

The add-on only accesses the currently active spreadsheet where the functions are used and performs read-only requests to Yahoo Financeâ„¢ to retrieve public market data. No personal user data is collected or stored.
